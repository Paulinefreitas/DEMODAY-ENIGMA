Um novo modo de criar suas senhas

Bem-vindo ao SenhaFácil

Agora é póssivel você criar uma senha criptografada apartir de uma palavra qualquer. Para criptografar click no botão criptografar, para voltar na palavra que você scolheceu, basta clicar em descriptografar. É necessário escolher um número de deslocamento para sua senha, o uso de caracteres maiúsculas e minúsculas devem ser utilizados. Divirta-se!
